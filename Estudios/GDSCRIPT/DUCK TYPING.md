
# definición 

Es una libertad o característica que te da los lenguajes dinámicos como python y GDScript que se puede definir con las siguiente frase que es clasica en este termino

**Si luce como pato, cuakea como pato y camina como pato, entonces es un pato**

Lo que nos quiere decir que una clase podria ser pasada como otra clase si cumple con los requerimientos minimo ya sea con los metodos o atributos. 

---

# *ejemplo *

```
Clase animal // clase base
Esta vivo

Funcion hablar

Clase perro hereda clase animal

Funcion hablar redefinir 
Imprimir   wau!! 


Clase auto 

Funcion hablar 
Imprimir honk!! 
```


---


Si en este ejemplo hacemos un vector de la clase animales y ocupamos la funcion hablar 
Y se encuentra algun objeto auto. 
No surgira ningún problema o error en la ejecución del codigo.
